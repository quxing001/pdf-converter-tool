### 第二阶段：接入支付（有用户后）

## Stripe（最推荐，最简单）

### 为什么选 Stripe？
- 🌍 全球通用
- ⚡ 申请只需 5 分钟
- 🎯 无需营业执照（个人可用）
- 💵 自动换汇（美元→人民币）
- 📊 完善的数据分析
- 🔒 安全可靠

### 申请步骤

1. **注册账号**
   - 访问：https://dashboard.stripe.com/register
   - 填写邮箱、密码
   - 选择个人账号

2. **完善信息**
   - 姓名
   - 地址（可用中文）
   - 电话号码
   - 身份信息（身份证/护照）

3. **获取 API 密钥**
   - 进入 Dashboard
   - Developers → API keys
   - 复制 Publishable key 和 Secret key

4. **接入代码**（我已经准备好）

**费用：**
- 交易手续费：2.9% + $0.30
- 无月费
- 无注册费
- 自动转账到银行卡

## 微信支付（面向中国用户）

### 申请条件
- ✅ 已认证微信公众号/小程序
- ✅ 营业执照（个体工商户也可以）
- ✅ 银行对公账户
- ✅ 保证金 ¥5,000

### 申请步骤
1. 登录微信支付商户平台
2. 提交营业执照、身份证
3. 绑定银行账户
4. 缴纳保证金
5. 等待审核（3-7 个工作日）

**费用：**
- 交易手续费：0.6%
- 保证金：¥5,000
- 年费：¥300

---

## 支付宝（面向中国用户）

### 申请条件
- ✅ 个人支付宝账号（个人版）
- ✅ 企业支付宝账号（企业版）
- ✅ 营业执照（企业版）

### 申请步骤
#### 个人版（推荐初期使用）
1. 下载"支付宝商家版"APP
2. 注册商家账号
3. 绑定个人银行卡
4. 实名认证
5. 生成收款二维码

#### 企业版（正式运营）
1. 访问支付宝开放平台
2. 创建应用
3. 提交营业执照
4. 等待审核（1-3 天）
5. 配置密钥

**费用：**
- 个人版：免费（但有交易限额）
- 企业版：0.6%

## 🚨 重要提醒

### 法规合规
- 个人收款：单笔 ¥5 万以下，每日 ¥20 万以下
- 企业收款：无限制
- 需要申报个人所得税（如盈利）

### 税务问题
- 个人收款：按"劳务报酬"或"经营所得"申报
- 企业收款：按企业所得税申报
- 建议月收入超过 ¥5,000 时咨询会计师

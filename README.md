# PDF 转换工具 - 自动运行赚钱系统 Demo

## 📋 项目说明

这是一个完整的 PDF 转换工具 demo，展示了如何创建一个可以自主运行的赚钱系统。

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 运行开发服务器
```bash
npm run dev
```

访问 http://localhost:3000 查看效果

### 构建生产版本
```bash
npm run build
```

## 📦 部署到 Vercel（免费）

1. 访问 [vercel.com](https://vercel.com) 注册账号
2. 点击 "Add New Project"
3. 导入这个项目的 GitHub 仓库
4. 点击 "Deploy"（Vercel 会自动检测 Next.js 项目）
5. 等待 1-2 分钟，网站就上线了！
